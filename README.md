## Install for windows
### Zend Framework
```
cd zend
composer update
composer install
composer serve
```

### Vue Cli
```
cd vue
npm install
npm install google-maps
npm run serve
```

### Line Messaging API
```
import database.sql
Edit -> index.php-> $conn = mysqli_connect('localhost', 'username', 'password','database') or die("ขออภัยค่ะ ขณะนี้ไม่สามารถติดต่อฐานข้อมูลได้ กรุณาเข้าชมใหม่ภายหลัง");
```
